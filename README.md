# yt-no-shorts

uBlock Origin filters to eradicate YouTube Shorts from desktop and mobile. (fuck shorts)

## Install

### Option 1: Subscribe (auto-updates)

1. Open uBlock Origin → **Filter lists** → **Import...**
2. Paste the raw URLs:

```
https://raw.githubusercontent.com/vonkoq/yt-no-shorts/master/desktop-filters.txt
```
```
https://raw.githubusercontent.com/vonkoq/yt-no-shorts/master/mobile-filters.txt
```

3. Click **Apply changes**

### Option 2: Manual (My Filters)

1. Open uBlock Origin → **My filters**
2. Copy and paste from [desktop-filters.txt](desktop-filters.txt) and/or [mobile-filters.txt](mobile-filters.txt)
3. Click **Apply changes**

## What it blocks

- Shorts in homepage, subscriptions, search, sidebar, channel pages
- Shorts navigation button
- Shorts sections and shelves
- Videos with `#shorts` in title
